QUICK START (30 MINUTES)
========================

1. Install Sentry
   $ npm install @sentry/nextjs

2. Create account at sentry.io
   - Sign up for free
   - Create Next.js project
   - Copy DSN

3. Add to .env.local
   NEXT_PUBLIC_SENTRY_DSN=https://xxx@yyy.ingest.sentry.io/123
   NEXT_PUBLIC_APP_VERSION=0.1.0

4. Initialize in src/app/layout.tsx
   import { initializeSentry } from '@/lib/monitoring'
   if (typeof window === 'undefined') {
     initializeSentry()
   }

5. Test health endpoint
   GET http://localhost:3000/api/health

6. Setup Uptime Robot
   - Go to uptimerobot.com
   - Add monitor: https://your-app.com/api/health
   - Enable Slack notifications
