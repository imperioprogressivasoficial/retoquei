================================================================================
                          ESTRUTURA DE COPY
================================================================================

Todos os documentos seguem esta estrutura:

1. HEADLINE IMPACTANTE
   → Despertar para o problema ou benefício

2. SUBHEADLINE
   → Explicar a solução brevemente

3. PROBLEMA (O QUE DÓI)
   → Fazer o leitor se identificar
   → Quantificar em $$$

4. SOLUÇÃO (COMO RESOLVE)
   → 3 passos simples
   → "Seu gerente consegue fazer"

5. BENEFÍCIOS (O QUE VOCÊ GANHA)
   → 5 benefícios principais
   → Foco em: receita, tempo, dados, competitividade

6. SOCIAL PROOF (NÚMEROS & DEPOIMENTOS)
   → Estatísticas reais
   → Depoimentos de clientes
   → Case studies

7. PREÇO (O QUANTO CUSTA)
   → 3 tiers (starter, profissional, premium)
   → Mostrar ROI (não o preço como problema)

8. GARANTIA (REMOVER RISCO)
   → 30 dias full retorno
   → 14 dias grátis sem cartão

9. CALL-TO-ACTION (O QUE FAZER)
   → "Começar teste grátis"
   → "Agendar demo"
   → "Ver estrutura completa"

10. FAQ / OBJEÇÕES (ANTECIPAÇÃO)
    → Responder dúvidas antes delas surgirem

================================================================================
                        DICAS DE IMPLEMENTAÇÃO
================================================================================

1. COMECE PEQUENO
   Não lance tudo de uma vez. Teste com 20-50 pessoas primeiro.
   Se funcionar (click > 10%), escala.

2. MANTENHA A CONSISTÊNCIA
   Use o mesmo headline em Google Ads, Email, LinkedIn.
   Repetição = memorização = conversão.

3. FOQUE NO ÂNGULO
   Você tem 3 ângulos principais:
   - RECEITA: "Aumente faturamento 40%"
   - TEMPO: "Zero trabalho manual"
   - COMPETITIVIDADE: "Saia à frente dos concorrentes"

   Escolha um. Martele-o em todos os canais.

4. TESTE RÁPIDO
   Faça A/B testing de headlines, CTAs, offers.
   Melhor resultado vence. Escala aí.

5. ACOMPANHE NÚMEROS
   CTR, conversion rate, CAC, LTV.
   Sem dados, você está no escuro.

6. ADAPTE À SUA VOZ
   O copy é um template. Adapte para seu tom.
   Se você é casual, fica casual.
   Se você é profissional, fica profissional.
   Mas mantenha a ESTRUTURA.
